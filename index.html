The World Wide Web - WWW

The web is a bunch of computers that communicate with each other via the Internet. When a person goes to a web page their computer sends a HTTP Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.

HTML

HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". This video gives a good overview.

Tags & Elements (inline vs block)

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them

Computers are stupid:

This refers to the idea that computers only do what they are "told." This means when coding explicit instructions need to be given for any scenario that may be encountered, luckily coding frameworks establish most of these "basic" instructions so they don't constantly need to be repeated.

The Language of the Web - HTML/CSS/Javascript

Lesson 2 introduces the structure of web pages. All pages are composed of HTML elements to establish their underlying "structure." Additionally web pages can utilize CSS or Cascading Style Sheets or Javascript.

CSS - Cascading Style Sheets

CSS is the paint on the walls. Through minimal amounts of code using CSS you can quickly transform the style of a website. CSS is stored in (dot).css files. Specifically this is referenced as increased maintainability.

JS - JavaScript

Java-script is the "garage door opener" for this web "home" we've been referencing. JS is a mechanism to accomplish many things rather easily. A prime example would be using a simple JS technique to allow the user to change the theme of a web page by loading other CSS properties.

Dont be a Square

All elements in HTML are rectangular in shape, even if they don't appear that way visually. The web browser builds/interprets a tree like structure of every page through utilization of the Document Object Model (DOM).

Avoid Repetition

Avoiding repetition in coding is essential to ensure ease of use and troubleshooting. Using portions of code to control multiple areas will expedite the customization process as well as make it much easier for others to interpret your code. This is especially possible through the use of CSS
